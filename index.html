数据分析 / Python · Tableau

TravelGo 旅游电商平台用户画像与商业仪表盘分析

工具栈：Python (Pandas, Matplotlib, Seaborn, Pyecharts) · Jupyter Notebook · Tableau · Excel

以 TravelGo 旅游电商平台为虚拟业务场景，通过对 1000 条用户行为数据 + 2080 条消费记录的清洗与分析，利用 Python 完成用户画像与复购率研究，并使用 Tableau 搭建商业仪表盘，实现多维数据可视化与业务洞察。

🔑 发现一
机票品类复购率最高，酒店与旅游套餐复购集中在首月，后续流失明显——建议针对首月后的用户精准推送优惠券。
🔑 发现二
优惠券整体覆盖率约 39%，四品类分布均匀；机票覆盖率最高达 42%，旅游套餐最低仅 35%，存在提升空间。
🔑 发现三
Top 20% 高消费用户偏好集中于科技数码（50%）和美食探店（33%），建议首页推荐向此方向倾斜。

📋 一、背景设定

作为 TravelGo 旅游电商平台的运营数据分析师，面临以下业务问题：

  • 用户流失加剧：近3个月高价值用户复购率下降15%
  • 优惠券效果存疑：市场部质疑优惠券投放是否覆盖了大部分商品
  • 内容策略调整：需要根据用户偏好优化APP首页内容推荐

📊 二、数据清洗

2.1 数据基本情况
数据全部成功加载： 用户行为数据形状： (1000, 8) 消费记录数据形状： (2080, 7)
2.2 处理缺失值 & 异常值

对缺失值采用删除策略；异常点击率（>0.08）视为异常数据予以处理；消费数据去重验证。

📈 三、数据分析

3.1 复购率数据分析

针对消费信息表中的用户编号、商品类别和消费日期进行复购率分析，发现机票品类复购率最高，酒店与旅游套餐复购集中在首月。
